# Machinery Leasing
A simple Machinery Leasing Application 
created in Spring Boot

## Run locally

### Requirements
1. JDK 17
2. Maven
3. Docker and docker-compose

### Steps:
1. Run
```shell
docker-compose up -d
```
2. Insert seed data running
```shell
make seed_data
```
3. Open the Project in your IDE 
with Maven support and 
execute the class 
`MachineryLeasingApplication.java`
